# Gulp New

Пустой настроенный на работу gulp проект с минимальным функционалом:Jquery, Bourbon mixins, Bootstrap_v4, Tether
## Задачи

### Новый проект

Для ничала работы с новым проектом:

```
npm i
gulp init
gulp
```

### Сборка Production

```
gulp build
```
